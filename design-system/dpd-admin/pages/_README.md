# Page-specific design overrides

Drop a file named after the route slug here to override rules from [`MASTER.md`](../MASTER.md).

**Example:** `drivers.md` overrides global rules when building `/drivers` pages.

**Hierarchy:**
1. `design-system/dpd-admin/pages/<page>.md` (if exists)
2. `design-system/dpd-admin/COMPONENTS.md`
3. `design-system/dpd-admin/TOKENS.md`
4. `design-system/dpd-admin/MASTER.md`

**Reference screens:** See [`../../references/`](../../references/) for visual ground truth.
